# Zhihu Plus 知乎阅读小助手 Chrome浏览器扩展

## 安装方式

方式一： 打开更多工具 → 扩展程序 → 打开开发者模式 → 加载已解压的扩展程序 → 选择 dist 目录

方式二：直接拖拽文件 `zhihu-plus.crx` 到 扩展程序页面的空白处。
## 功能说明

  1. 暗色极简阅读模式，不刺激眼睛，更适合摸鱼；

  2. 去除多余的东西，比如说官方或用户插入的广告之类的;

  3. 纤细优雅的滚动条。
